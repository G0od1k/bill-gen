List of all short codes:
{date} – Current date
{time} – Current time
{hr} – Horizontal rule. You can specify the symbol(s) used for generating the horizontal rule after the code. For example, {hr}= will create a rule with equal signs.
{bill} – Placeholder for all positions in the bill with their prices
{sum} – Total sum of the bill
{t}text – Aligns the text to the right, which can be accompanied by text on the left
{c}Text – Centers the text
{img}id:size – Used to insert an image in image mode. For more details on usage, refer to the "Images" subpage

Positions

Images

To use images in the bill, add a line to the bill in the format {img}id:size, where you should replace id with the ID of your image. The size can be specified in one of the following formats: 75% – percentage of the bill's width, 10ch – width in characters, 100 – width in pixels.
